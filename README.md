
  # Athlete Performance Tracker

  This is a code bundle for Athlete Performance Tracker. The original project is available at https://www.figma.com/design/vn6CGGdvfOucgrEl50T40X/Athlete-Performance-Tracker.

  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.

  ## Feedback from seminar:
  ### Creating a journal entry
  A bit confusing at the start since there are two (+) buttons (one big in the middle whenever no entries exist) → Maybe remove or make a bit more clear somehow.

  There could maybe be predefined options for “mood” as athletes may just want to quickly choose it if they’re tired or something → Make mood a dropdown selection, maybe leave room for manual input.

  Reflective questions - Could include a “Note” field where the athlete can write anything that they thought was important but wasn’t covered by the guide questions.

  When an entry is done - Make the overview/summary of the entry more clear → add separations, bold, etc. to more clearly separate the answers and make it more readable.
  
  ### Goals page:
  Likes that there are examples in the text fields - makes it easier to understand as a first time user.

  Maybe switch order of category and goal as you maybe “know” the category first. Also depending on which category is chosen (performance/fitness/skill/…) make the example text in the text field change based on selected category.
  
  ### Home page:
  A bit too cluttered/much having both the big navigation buttons AND navigation bar → Keep the navigation bar on other screens but remove it from the home page.

  The home button in the navigation bar - Either move it to the left or make it more highlighted compared to the other buttons so that it is easier to find. 
  A profile button would be nice to have. 

  Video and Video Learning button a bit confusing on what the difference between them is unless you read the smaller text. Either make it more clear or maybe move that functionality to the video page.

  Likes the buttons and that they have different colours - makes it more easy to differentiate. 
  
  ### Overall:
  While the app gives good tools to analyse and reflect - it doesn’t really give any new insights as you only see/get what you yourself fill in text fields and upload to the app.
  
  Maybe some AI to give you insights on your reflection?
  Discussion with your coach → Will be added as comments on videos where the athlete and the coach can discuss and analyse the athlete’s performance.
